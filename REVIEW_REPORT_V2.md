## 🔧 本次修复的Bug

### Bug #1: SQL 字段名不匹配

**问题**: `lib/socket.ts` 中查询 `plugin_butler_scheduled_reminders` 表时使用了 `creator_id`，但实际表字段是 `created_by`

**错误日志**:
```
[FamilyButler] 查詢提醒事項失敗(可能表不存在): SqliteError: no such column: creator_id
```

**修复**:
- 文件: `lib/socket.ts`
- 行号: 593
- 修改: `creator_id` → `created_by`

**状态**: ✅ 已修复
