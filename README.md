# 🏠 家族中心网站

一个专为全年龄段设计的家族社交平台，从小学三年级学生到七十多岁老人都能轻松使用。

## ✨ 核心功能

### 🎯 面向全年龄段的易用性设计
- **超大字体**：所有文字都使用更大字号，老人小孩都能轻松看清
- **超大按钮**：最小点击区域 60px，避免误触，适合手指不灵活的用户
- **超大输入框**：20px 字号输入，打字更轻松
- **高对比度**：色彩对比度符合 WCAG 无障碍标准，视力不好也能看清
- **简单清晰的界面**：没有复杂的操作，每一步都有明确提示
- **图标+文字双标识**：所有功能都有图标和文字双重提示，不认字也能看懂

### 👥 用户系统
- ✅ 邮箱+密码注册登录，简单易操作
- ✅ 自动记住登录状态 7 天，不用每次都输密码
- ✅ 个人头像和信息管理

### 👨‍👩‍👧‍👦 家族群组系统
- ✅ 创建专属家族群组
- ✅ 邀请制加入，需要管理员同意
- ✅ 不同家族数据完全隔离，保护隐私
- ✅ 管理员/成员角色管理
- ✅ 待处理邀请提醒，一目了然

### 📢 公告栏
- ✅ 管理员可以发布家族重要通知
- ✅ 所有成员都能看到最新公告
- ✅ 按时间倒序排列，最新的在最上面

### 💬 留言板
- ✅ 所有成员都可以发表留言
- ✅ 支持长文本，分享生活点滴
- ✅ 显示发布者头像和时间
- ✅ 按时间倒序排列

### 💬 即时聊天（开发中）
- ✅ 家族成员实时在线聊天
- ✅ 消息实时推送
- ✅ 历史消息记录

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装依赖
```bash
cd family-website
npm install
```

### 开发模式运行
```bash
npm run dev
```
访问 http://localhost:3000 即可查看网站

### 生产环境构建
```bash
npm run build
npm start
```

## 📱 移动端优先
网站完全响应式设计，手机、平板、电脑都能完美适配，主要针对手机使用优化：
- 底部导航栏，大拇指就能点到
- 所有元素都适合触屏操作
- 支持横屏竖屏切换

## 🔐 安全特性
- 密码使用 bcrypt 加密存储
- JWT 身份验证，安全可靠
- 家族数据完全隔离，不同家族之间看不到对方数据
- 所有接口都有权限验证，防止越权访问

## 🛠 技术栈
- **前端**：Next.js 15 + React 19 + TypeScript
- **样式**：Tailwind CSS
- **数据库**：SQLite (文件型数据库，无需额外安装)
- **认证**：JWT + bcrypt
- **实时通信**：Socket.IO (聊天功能)

## 🎨 设计理念
### 针对老年人的优化
- 没有复杂的手势操作，都是简单的点击
- 没有闪烁、动画等可能引起眩晕的效果
- 所有可点击元素都有明确的边框和反馈
- 错误提示用大字体和表情符号，清晰易懂
- 操作步骤少，尽量一步到位

### 针对儿童的优化
- 界面色彩明亮友好
- 图标生动有趣
- 操作简单，容易理解
- 没有复杂的表单

## 📝 使用说明
1. **注册账号**：输入姓名、邮箱、密码即可注册
2. **创建家族**：登录后点击"创建家族"，输入家族名称即可
3. **邀请成员**：作为管理员可以邀请家人加入
4. **发布公告**：管理员可以在公告栏发布重要通知
5. **留言互动**：在留言板分享生活点滴
6. **实时聊天**：和家族成员在线聊天
