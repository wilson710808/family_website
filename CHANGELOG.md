# 更新日志

## v1.2.0 (2026-03-21)

### ✨ 新增功能
- 🎁 **生日/纪念日提醒插件** (完全可插拔设计)
  - 支持生日、结婚纪念日、自定义提醒
  - 每年自动循环提醒，可设置提前N天提醒
  - 独立管理页面 `/plugins/birthday`
  - 首页仪表板显示即将到来的提醒
  - 通过环境变量 `PLUGIN_BIRTHDAY_REMINDER=false` 可完全禁用

- 🖼️ **家族相册插件** (完全可插拔设计)
  - 支持创建多个相册分类
  - 照片上传、存储、展示
  - 支持点赞和评论互动
  - 首页仪表板显示最新上传照片
  - 独立访问页面 `/plugins/album`
  - 通过环境变量 `PLUGIN_FAMILY_ALBUM=false` 可完全禁用

- ⭐ **贡献积分星级系统增强**
  - 新增家族关系称谓字段（爸爸、妈妈、儿子等）
  - 新增贡献积分和自动星级计算：
    - 1⭐ = 新成员 0-99 积分
    - 2⭐ = 创建者/活跃成员 100-299 积分
    - 3⭐ = 核心成员 300-599 积分  
    - 4⭐ = 资深成员 600-999 积分
    - 5⭐ = 顶流贡献 1000+ 积分
  - 家族创建者默认二星起步（符合设计）
  - 自动计算和更新星级

### 🔓 需求变更
- 🚀 **移除注册和账户检查功能**
  - 删除注册页面和注册API
  - 所有访客自动以系统管理员身份登录
  - 移除登录界面和登出按钮
  - 现在无需注册，打开网站直接使用

### 🏗️ 架构改进
- 采用**可插拔插件架构**，新功能都以插件形式加入
- 插件独立目录、独立API路由、独立数据表（`plugin_`前缀）
- 通过环境变量控制启用/禁用，不需要修改核心代码
- 插件数据表独立，禁用后可完整删除不影响核心数据

### 🔐 数据库变更
- `family_members` 表新增字段：
  - `relationship` - 家族关系称谓
  - `contribution_points` - 贡献积分
  - `contribution_stars` - 贡献星级
- 新增插件独立表：
  - `plugin_birthday_reminders` - 生日提醒
  - `plugin_birthday_settings` - 提醒设置
  - `plugin_album_albums` - 相册
  - `plugin_album_photos` - 照片
  - `plugin_album_likes` - 点赞
  - `plugin_album_comments` - 评论

## v1.1.0 (2026-03-15)

### ✨ 新增功能
- 🔧 修复登录后页面跳转问题，现在登录成功后会正确跳转到首页
- 👤 注册时支持选择头像，从9个预设头像中选择，方便家人识别
- 🔢 推荐码功能：
  - 每个家族创建时自动生成唯一的6位推荐码
  - 新用户注册时输入推荐码即可自动申请加入对应家族
  - 家族详情页显示推荐码，支持一键复制
  - 简化邀请流程，老人小孩也能轻松操作
- 📱 进一步优化移动端体验，继续加大字体和按钮尺寸，适合全年龄段使用

### 🔐 数据库变更
- users表新增 `referral_code` 和 `referred_by` 字段支持推荐关系
- families表新增 `referral_code` 字段存储家族推荐码
